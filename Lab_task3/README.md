Assignment Title:
Study Revision & Mock Test Management System

Description:
This project is a static website made using HTML and CSS.
The purpose of this system is to help students create learning notes,
plan revision dates, and view mock test details.
This project does not include backend or JavaScript.

Pages:
1. Home Page
2. Login Page
3. Dashboard Page
4. My Notes Page

Technologies Used:
HTML5
CSS3

Folder Structure:
index.html
login.html
dashboard.html
list.html
css/style.css
