# Interactive Product Card and Registration Form

## Description
This project is a simple web page built using HTML, CSS, and JavaScript.
It includes a product card and a user registration form with validation and UI features.

## Features
- Product card with Add to Cart message
- Image fallback handling
- User registration form
- Form validation with error messages
- Password show and hide option
- Dark mode and light mode
- Theme saved using localStorage
- No page reload on submit

## How to Run
1. Download or clone the project folder
2. Open `index.html` in any modern web browser
3. No installation required
