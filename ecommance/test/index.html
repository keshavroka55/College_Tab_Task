Key Features

Mobile Layouts Ready

By using mobile layouts, you can build a mobile-friendly website and bring your customers a great mobile shopping experience.

SEO Optimization

The theme is made with customers in mind. It was coded using the very latest practises to improve performance.

Multilingual RTL Support

Work with any languages installed in your theme. Include RTL languages support as Arabic, Hebrew, Persian...

Multistore Compatible

The theme is compatible with multistore site. With the theme, you can create your own marketplace and allow vendors to sell easily.

Fully Responsive

Integrating responsive function that gives your online store a beautiful interface on any high resolution devices or mobile and tables.

Google Fonts

The theme integrates 800+ Google Webfonts, offering you a large variety of typefaces to choose from when customizing the theme.

Sample Data

We included the quickstart package in this theme. Just a few clicks you can import easily CMS blocks and pages with sample content from the demo.

Megamenu [Drag & Drop]

Our Drag & Drop Mega Menu is the perfect choice for large menus. You can set up columns and rows, use icons and images easily.

Ajax cart

You can select product to buy without leaving current page, instantly add product to basket. Increase user experience and sales.

NEW TECHNOLOGY

Tara is built with new web technology: Bootstrap 4, Font Awesome 4, LESS, CSS3, HTML5 that makes the theme more powerful and flexible.

Unlimited Color

There are 6 couple colors available for your choose from in them Admin Panel, you can also make your own color theme easily.

Easy Customizable

Easy change theme options such as homepage layout, color, menu styles, etc in admin panel without touching any code lines.
